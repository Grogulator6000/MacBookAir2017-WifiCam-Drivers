This is a WIP repo.

JUST FOR MACBOOK AIR 2017 (or another laptop which uses the same wifi hardware (Broadcom BCM4360))

Currently, it only deals with getting wifi working. I haven't been able to find a reliable source for the facecam (facetimehd) driver (it's distributed as a zip file, but the hosts take it down).

So, yes wifi, no facecam (just yet) with this script.

--
Instructions:

Get a wired connection (I use USB-tethering with my phone).
Make sure to have non-free-firmware enabled in /etc/apt/sources.list
Update and Upgrade apt
Run the script (chmod u+x if it doesn't execute)
